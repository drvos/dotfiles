Installation
------------
The following tools are required:
- git
- curl
- tmux
- zsh
- fzf
- vim

```sh
git clone https://github.com/drvos/dotfiles.git 
cd dotfiles
./install.sh
```
The configuration files are symlinked to the home directory.
